Basketball court — Full procedure (HTML + CSS)

This single file includes: a responsive SVG court drawing and step-by-step instructions. You can edit the SVG to change colors, sizes, or add logos.

Quick steps

  1. Copy the entire file and save as basketball-court.html.
  2. Open the file in a browser (double-click or drag into a tab).
  3. To customize: open the file in a text editor and change the colors under the :root variables.
  4. To change court proportions or add markings, edit the SVG shapes inside the <svg> element.
  5. For interaction (drag a ball, animate), add a small JavaScript block — but this example uses only HTML & CSS.

Why SVG?

SVG keeps the court crisp at any size, is easy to style, and accessible. The viewBox (1000x500) gives a 2:1 court ratio — consistent with a basketball court layout.

Tips & next steps

  • Make the floor look more realistic: add more subtle gradients, wood plank lines using thin rects, or a background image.
  • Add labels (team logos) using <image> inside SVG (remember cross-origin restrictions if linking remote images).
  • Create an interactive practice tool by adding a bouncing ball and simple physics in JavaScript.
